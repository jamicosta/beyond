# BEYOND - Site de Marca de Roupas

Um projeto web simples e amigável para iniciantes para uma marca de roupas moderna. Este projeto demonstra habilidades fundamentais de desenvolvimento web, incluindo estrutura HTML, estilo CSS e interatividade JavaScript.

## Visão Geral do Projeto

**BEYOND** é um site de marca de roupas fictício projetado para ajudar a aprender o básico do desenvolvimento web. O site inclui navegação, vitrine de produtos, formulário de contato e funcionalidade de compras.

## Arquivos Incluídos

- **index.html** - Estrutura HTML principal com seções para herói, sobre, produtos e contato
- **styles.css** - Estilo com design responsivo e layout moderno
- **script.js** - JavaScript para interatividade e manipulação de formulários
- **README.md** - Este arquivo

## Recursos

### Estrutura HTML
- Elementos HTML semânticos (header, main, footer, section, nav)
- Menu de navegação com rolagem suave
- Múltiplas seções de conteúdo (herói, sobre, produtos, contato)
- Cards de produtos com botões
- Formulário de contato

### Estilo CSS
- Layout de grade responsivo para produtos
- Fundos em gradiente e efeitos de hover
- Design amigável para dispositivos móveis com media queries
- Transições suaves e animações
- Esquema de cores moderno (roxos e cianos)

### Funcionalidade JavaScript
- Sistema de carrinho de compras (armazena itens em um array)
- Manipulação de envio de formulário
- Navegação com rolagem suave
- Logging em console para depuração
- Animações acionadas por scroll
- Confirmações de interação do usuário

## Começando

1. Abra `index.html` no seu navegador da web
2. Explore as diferentes seções clicando nos links de navegação
3. Clique nos botões "Comprar Agora" ou "Adicionar ao Carrinho" para interagir com o site
4. Tente enviar o formulário de contato
5. Abra as Ferramentas do Desenvolvedor (F12) para ver os registros do console

## Conceitos de Aprendizagem

Este projeto cobre:

- **HTML**: Estrutura semântica, formulários, botões, links
- **CSS**: Flexbox, Grid, Media Queries, Gradientes, Transições
- **JavaScript**: Variáveis, Arrays, Funções, Event Listeners, Manipulação do DOM
- **UX/UI**: Navegação, Estados de hover, Design responsivo

Funciona em todos os navegadores modernos:

- Safari

## Como Usar

### Adicionando Produtos ao Carrinho
1. Clique em "Adicionar ao Carrinho" em qualquer produto
2. Verifique o console do navegador (F12) para ver o conteúdo do carrinho

### Enviando Mensagens
1. Preencha o formulário de contato
2. Clique em "Enviar Mensagem"
3. Verifique o console para os dados do formulário

### Navegação
1. Clique em qualquer item do menu de navegação
2. A página rola suavemente para a seção
3. Verifique o console para o status da página
