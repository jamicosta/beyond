# BEYOND - Clothing Brand Website

A simple, beginner-friendly web project for a modern clothing brand. This project demonstrates fundamental web development skills including HTML structure, CSS styling, and JavaScript interactivity.

## Project Overview

**BEYOND** is a fictional clothing brand website designed to help learn the basics of web development. The site includes navigation, product showcase, contact form, and shopping functionality.

## Files Included

- **index.html** - Main HTML structure with sections for hero, about, products, and contact
- **styles.css** - Styling with responsive design and modern layout
- **script.js** - JavaScript for interactivity and form handling
- **README.md** - This file

## Features

### HTML Structure
- Semantic HTML elements (header, main, footer, section, nav)
- Navigation menu with smooth scrolling
- Multiple content sections (hero, about, products, contact)
- Product cards with buttons
- Contact form

### CSS Styling
- Responsive grid layout for products
- Gradient backgrounds and hover effects
- Mobile-friendly design with media queries
- Smooth transitions and animations
- Modern color scheme (purples and cyans)

### JavaScript Functionality
- Shopping cart system (stores items in an array)
- Form submission handling
- Smooth scroll navigation
- Console logging for debugging
- Scroll-triggered animations
- User interaction confirmations

## Getting Started

1. Open `index.html` in your web browser
2. Explore the different sections by clicking navigation links
3. Click "Shop Now" or "Add to Cart" buttons to interact with the site
4. Try submitting the contact form
5. Open browser Developer Tools (F12) to see console logs

## Learning Concepts

This project covers:

- **HTML**: Semantic structure, forms, buttons, links
- **CSS**: Flexbox, Grid, Media Queries, Gradients, Transitions
- **JavaScript**: Variables, Arrays, Functions, Event Listeners, DOM Manipulation
- **UX/UI**: Navigation, Hover states, Responsive design

## Browser Compatibility

Works on all modern browsers:
- Chrome/Edge
- Firefox
- Safari

## How to Use

### Adding Products to Cart
1. Click "Add to Cart" on any product
2. Check the browser console (F12) to see the cart contents

### Sending Messages
1. Fill out the contact form
2. Click "Send Message"
3. Check the console for form data

### Navigation
1. Click any navigation menu item
2. Page smoothly scrolls to the section
3. Check console for page status
